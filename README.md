# Music Popularity Prediction Using Machine Learning

## Overview
This project explores how audio features, metadata, and social engagement indicators influence music popularity using machine learning models and interpretable AI techniques.

## Technologies Used
- Spotify Web API
- Python
- Pandas
- NumPy
- Scikit-learn
- XGBoost
- SHAP
- Matplotlib
- Jupyter Notebook

## Features
- Spotify API integration for retrieving popularity scores
- Data preprocessing and cleaning
- Multimodal feature analysis
- Music popularity prediction
- Class imbalance handling
- SHAP interpretability analysis
- Model comparison and evaluation

## Models Used
- Logistic Regression
- XGBoost

## Dataset
The project uses a music dataset containing Spotify audio features, metadata, and social engagement indicators.

## Results
The analysis demonstrates that combining audio, metadata, and social engagement features improves music popularity prediction and interpretability.
